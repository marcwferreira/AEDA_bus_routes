Detailed Description

This class is a system to store information while performing a search of a path on the graph

Parameters
codeThis is the code of the bus stop it stores information from
lineCodeThis is the code of the bus line we are using in this stop
distanceThis is the distance from this bus top to the start the user selected
previousThis is the bus stop previous to getting to this one of the path
visitThis stores if this bus stop was already visited or not
linesChangedThis stores the lines we used to get from the start (user selected) to this bus stop
zonesThis is the zones we used to get from the start (user selected) to this bus stop
